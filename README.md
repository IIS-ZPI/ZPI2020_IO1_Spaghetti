# ZPI2020_IO1_Spaghetti

Workflow umożliwia automatyczne zbudowanie projektu, zainstalowanie wymaganych bibliotek i wykonanie testów. 
Aktualnie dzieje się to przy użyciu jednej wersji języka Python (3.9) i działa po każdym pull requestcie na develop/release. W planie jest tworzenie tagów po każdym PR do gałęzi release.
Akcje wykonują się z wykorzystaniem wewnętrznego narzędzia GitHuba, a logi z każdego uruchomienia dostępne są w zakładce "Actions" projektu.
