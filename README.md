# ZPI2020_IO1_Spaghetti

Workflow umożliwia automatyczne zbudowanie projektu, zainstalowanie wymaganych bibliotek, wykonanie testów i automatyczne stworzenie taga. 
Aktualnie dzieje się to przy użyciu jednej wersji języka Python (3.9) i działa po każdym pushu na gałąź release.

Akcje wykonują się z wykorzystaniem wewnętrznego narzędzia GitHuba, a logi z każdego uruchomienia dostępne są w zakładce "Actions" projektu:
https://github.com/IIS-ZPI/ZPI2020_IO1_Spaghetti/actions

Backlog:
https://github.com/IIS-ZPI/ZPI2020_IO1_Spaghetti/projects/1

Raporty z testów w formie plików .docx znajdują się na głównej gałęzi, w folderze test_reports katalogu projektowego: 
https://github.com/IIS-ZPI/ZPI2020_IO1_Spaghetti/tree/develop/test_reports
